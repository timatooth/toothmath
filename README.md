## Installation

The package can be installed
by adding `toothmath` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:toothmath, "~> 0.1.0"}
  ]
end
```

or from a script

```elixir
Mix.install([
  {:toothmath, "~> 0.1.0"}
])
```
